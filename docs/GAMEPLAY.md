# Ressources
 - Gold
 - Elexir
 - Black Elexir

# Start a game

- 4 players in every corners of the map.
- 1 Hotel at the start

# End game

- Destroy all the Hotels of the map

# Buildings

- Hotel: center of the village, can be upgraded to unlock more buildings

For the next buildings, you can upgrade them by clicking on them.

- Elexir Pump: ressource, can be upgraded to get more Elexir
- Black Elexir Pump: ressource, can be upgraded to get more Black Elexir
- Gold Mine: ressource, can be upgraded to get more Gold
- Caserne: See [Caserne](##Caserne)
- Canon: ground defence
- Archer's tower: ground defence
- Labo: Upgrade troops

## Caserne

- Create troops:
  - Barbarian
  - Archer
  - Giant
- Limit troops number
